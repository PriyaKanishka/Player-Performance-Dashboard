# ⚽ Premier League Player Performance Dashboard – Tableau | SQL | Excel

## 🔍 Project Overview
This Player Performance Dashboard analyzes and compares Premier League 
defender statistics to evaluate individual player contributions across 
key performance metrics. The dashboard focuses on Trent Alexander-Arnold 
of Liverpool F.C. benchmarked against top defenders in the league.

## 📁 Dataset
The dataset includes:
- Player demographics and club information
- Minutes played and assists
- Progressive passes per 90 minutes
- Shot creating actions per 90 minutes
- Tackles won and interceptions per 90 minutes
- Ball progression via carries per 90 minutes

## 🔑 KPIs Tracked
- Progressive Passes per 90
- Shot Creating Actions (SCA90)
- Tackles Won per 90
- Interceptions per 90
- Ball Progression via Carries per 90

## 📈 Key Insights
- Trent Alexander-Arnold leads all defenders in Progressive Passes 
  per 90 with 250+ and Shot Creating Actions at 3.820
- Lewis Hall ranks second in Shot Creating Actions at 3.580 
  followed by Aaron Cresswell at 2.940
- Trent sits above median in both Tackles Won and Interceptions 
  confirming his dual contribution in attack and defense
- Ball progression via carries shows Trent above the median 
  in carries into the final third

## 💡 Recommendations
- Trent Alexander-Arnold is statistically the most progressive 
  defender in the Premier League combining elite attacking output 
  with solid defensive metrics
- Teams defending against Liverpool should account for his 
  exceptional pass progression and shot creation threat

## 🛠 Tools & Technologies
- Tableau — interactive dashboard and visualizations
- SQL — data extraction and filtering
- Excel — data preparation and cleaning

## 📊 Dashboard Features
- Player profile card with club badge and key stats
- Progressive Passes per 90 bar chart comparing top defenders
- Shot Creating Actions per 90 bar chart
- Defensive profile scatter plot — Tackles Won vs Interceptions
- Ball Progression via Carries scatter plot with median benchmarks

## 📌 Conclusion
This dashboard provides a comprehensive performance profile of Premier 
League defenders enabling scouts, analysts, and fans to benchmark 
individual players across attacking and defensive metrics in a single view.
